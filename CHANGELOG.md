- 🔊 语音发音功能（小米 TTS API）

- 📱 响应式设计（支持手机和平板）

- Flutter 3.x 框架
- Riverpod 状态管理
- GoRouter 路由管理

## 版本说明

### 版本号格式
- **主版本号**：重大功能更新或不兼容变更
- **次版本号**：新功能添加，向下兼容
- **修订号**：Bug 修复和小改进

### 更改类型
- **新增**：新功能
- **变更**：现有功能的更改
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：Bug 修复
- **安全**：安全相关的更改

## 如何更新

### Android
1. 下载最新版本的 APK
2. 卸载旧版本（可选，数据会保留）
3. 安装新版本

### 从源码构建
```bash
git pull origin master
flutter pub get
flutter build apk --release
```
